1-bit gates

  • NOT Gate: bit flip
    or A ¬ B A \lnot B A¬B
  • FANOUT: copies the bit

2-bit gates

  • AND: 1 if A and B are 1, else 0
    or A ∧ B A\land B A∧B
    Equivalent to bit multiplication in 1-bit

  • OR: 1 if A or B are 1, else 0
    or A ∨ B A\lor B A∨B
    Schematically: Rectangle with two inputs, one output, curved arrow head on output side, curved indent on input side

  • XOR: 1 if either but not both A or B are 1, else 0
    or A ⊕ B A\oplus B A⊕B

  • NAND: 1 if A and B are not 1, else 0
    or ¬ ( A ∧ B ) \lnot(A\land B) ¬(A∧B)

Reversibility

Given an output, you can determine the input

  • Reversible gate: preserves all information
  • Non-reversible gate: loses some information

In quantum computing, all gates have to be reversible, since information has a physical meaning; if you lose information, that energy dissipation destroys your computation

Is NOT reversible?

Output 1 → \rightarrow → Input 0
Output 0 → \rightarrow → Input 1
Yes, it is reversible.

Is AND reversible?

Output 1 → \rightarrow → Input (1 1)
Output 0 → \rightarrow → Input (0 0), (0 1), or (1 0)

  • Cannot definitively identify the input in the case where the output is 0

No, it is not reversible.
